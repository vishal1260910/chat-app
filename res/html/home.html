Welcome to SimpleChat

SimpleChat is a modern, real-time web chat application built using Spring Boot. It demonstrates full-stack functionality with real-world tech like authentication, file handling, WebSockets, and responsive UI.

Key Features

  • User Authentication:
    • Session-based login
    • ‚ÄúRemember me‚Äù with cookies
    • Basic HTTP authentication
  • Secure self-signed HTTPS
  • Real-time WebSocket messaging
  • Upload and download file support
  • Data persistence with JPA (Hibernate)
  • Smart login redirection
  • File caching using modification dates
  • Idle session expiration after 1 hour
  • In-memory session storage for speed

User Interface Highlights

  • Fully responsive design
  • Interactive modals
  • Toast notifications
  • Real-time WebSocket updates
  • AJAX/HTTP Request usage

How to Use

  • Build: mvn package
  • Run: java -jar target/simplechat-0.0.1-SNAPSHOT.jar
  • Open: https://localhost:62600/
  • Sign up at least two user accounts
  • Start chatting!

What You Can Do

  • Send real-time text messages
  • Upload files to chat
  • Click file-messages to download
